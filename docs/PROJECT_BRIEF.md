# LandRegistry26 project brief

## Aim

Create a weekly automated UK property sales intelligence website using HM Land Registry Price Paid Data.

## Initial scope

- BN postcode prefix first.
- Standard Price Paid transactions by default.
- Weekly highlights, monthly observations, annual observations and area explorer.
- Designed to expand later to all UK/England & Wales regions or user-defined postcode prefixes.
